🪣 Bitbucket 가이드

🔀 Pull Request

Pull Request(PR)는 "내 브랜치를 main에 합쳐 주세요"라고 요청하며 팀의 리뷰를 받는 과정이에요.

1. 브랜치 올리기

먼저 작업한 브랜치를 Bitbucket에 올립니다.

$ git push origin feature/login

2. Pull Request 만들기

저장소 화면 왼쪽 메뉴 Pull requests → Create pull request를 누릅니다. 합칠 방향을 정해요: feature/login → main.

📷 여기에 "Create pull request" 화면 스크린샷

3. 제목 · 설명 쓰기

무엇을 왜 바꿨는지 적고, 리뷰어(Reviewers)를 지정한 뒤 Create를 누릅니다.

💡 설명에 "어떻게 테스트했는지"를 적으면 리뷰어가 훨씬 빨리 확인할 수 있어요.

4. 리뷰 → 머지

리뷰어가 코멘트를 남기면 수정해서 같은 브랜치에 다시 push하면 PR에 자동 반영돼요. 승인(Approve)을 받으면 Merge 버튼으로 main에 합칩니다.

5. 마무리

머지가 끝나면 내 컴퓨터에서도 최신 main을 받아옵니다.

$ git switch main
$ git pull origin main

축하해요! 🎉 이제 Bitbucket으로 팀과 협업하는 전체 흐름을 익혔습니다.
