🪣 Bitbucket 가이드

🔁 기본 Git 흐름

코드를 바꾼 뒤 Bitbucket에 올리는 가장 기본적인 4단계를 익혀요: 확인 → 담기 → 기록 → 올리기.

1. 무엇이 바뀌었는지 확인 (status)

$ git status

수정·추가된 파일 목록이 빨간색으로 보여요.

2. 올릴 변경사항 담기 (add)

$ git add .

.은 "바뀐 것 전부"라는 뜻이에요. 특정 파일만 담으려면 파일명을 적습니다.

3. 변경 내용 기록하기 (commit)

$ git commit -m "로그인 화면 추가"
💡 -m 뒤 메시지는 "무엇을 왜 바꿨는지" 나중의 나와 팀이 알아볼 수 있게 적어요.

4. Bitbucket에 올리기 (push)

$ git push origin main

이제 Bitbucket 저장소에서 방금 올린 내용을 확인할 수 있어요.

팀의 최신 코드 받기 (pull)

다른 팀원이 올린 변경을 내 컴퓨터로 가져옵니다. 작업 시작 전에 습관적으로 실행하세요.

$ git pull origin main
